To use the "decayingTurbulenceInflowGenerator" boundary condition type, 
you must install LEMOS-6 before running these simulations.

To download LEMOS-6, please visit here: 
    https://github.com/danhnam11/LEMOS-6

To run simulation:
    1. type blockMesh to generate mesh
    2. type realFluidFoam to run simulation 

Enjoy OpenFOAM.
